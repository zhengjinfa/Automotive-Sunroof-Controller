🌟 Project Overview

This repository hosts the hardware verification package for a 1-chip Car Sunroof Controller Solution. It is designed to replace traditional imported MCU solutions used in automotive sunroof systems.
Our goal is to provide the automotive industry with a high-reliability, low-cost, and fully localized alternative to reduce BOM costs and mitigate supply chain risks.

💡 Core Advantages

Why choose this solution? (Compared to traditional multi-chip/import chip 方案)

📉 Extreme Cost Reduction: BOM cost is optimized for high-volume production. Significantly lower than equivalent imported solutions.

🚗 Car Grade Reliability: Passes rigorous automotive tests (Temperature, Vibration, EMC, ISO 16750). Operates at -40°C to +125°C.

🔋 High Integration: All-in-One single chip design. Integrates Power Management, LIN/CAN Communication, and Motor Driver. No need for external auxiliary chips.

🌏 Stable Supply Chain: 100% Localized production. No shipping delays, tariffs, or geopolitical risks.

🔧 Technical Features (High Level)

Power Supply: 12V/24V compatible with Surge/Load Dump protection.
Driver: Single Relay / Dual Relay / Sunroof + Shade control supported.
Communication: LIN 2.0 / CAN Bus interface.
Protection: Anti-pinch, Over-current protection, Clamping detection.
Wake-up: IGN Wake & Key detection circuit.
